/*********************************************************************************************
函数名：毫秒级CPU延时函数
调  用：DELAY_MS (?);
参  数：1~65535（参数不可为0）
返回值：无
结  果：占用CPU方式延时与参数数值相同的毫秒时间
备  注：应用于1T单片机时i<600，应用于12T单片机时i<125
/*********************************************************************************************/
void DELY_MS(uint T)
{
	uint i;
	while(T-- !=0)	
	{
		for(i=0; i<600; i++);
	}
}
/*********************************************************************************************/
